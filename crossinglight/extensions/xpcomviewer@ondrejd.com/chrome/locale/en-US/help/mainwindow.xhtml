Introduction

Main &appFullName;'s window consists of few parts which are described below.

Main menu

Hlavní menu má tuto strukturu:

  • File
    • Reports
      • Save as XML - save report as XML document
      • Save as HTML - save report as HTML document
      • Save as TXT - save report as text document
    • Close - close &appFullName;
  • Show
    • Bookmarks - show/hide bottom bookmarks pane
    • Fullscreen - toggle fullscreen mode
    • LXR Search - show/hide LXR search page in browser's pane
    • JavaScript - show JavaScript console
    • Preferences - show preferences dialog
  • Help
    • Help - show this help
    • Home page - opens &appFullName;'s home page in your default browser
    • About - shows dialog about author and conributors of this extension

Main toolbar

  • Reports - save report from currently displayed items on the main tree into file in selected format.
    More informations you can find in caption Reports on page Using &appFullName;.
  • Add bookmark - add bookmark to selected tree item
  • Components.classes - show all available classes
  • Components.interfaces - show all available interfaces
  • Components.results - show all available results
  • LXR Search - show search page of default LXR server
  • Help - show this help
