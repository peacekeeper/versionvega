=== XBL JavaScript Shell ===

==== Description ====

This is a modification of JavaScript Shell which originally came from extension
"Extension Developer Tools" for the Mozilla Firefox (and others) [1].
This modification is heavilly rewriten 1.4 version of it.

===== Changes =====

 - full localization (cs-CZ, en-US)
 - bundled with help (modification of original help [3])
 ? rewritten as XBL component (all implemented in XUL, no HTML)
 ? upgrade to latest JSShell version (1.5)
 ? shell CSS customized
